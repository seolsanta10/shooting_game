# 비행 시뮬레이션 설정 가이드

## 생성된 오브젝트
- **지구**: 스케일 (50, 50, 50) - 비행기의 100배 크기
- **Player**: 스케일 (0.5, 0.5, 1) - 비행기

## 컴포넌트 설정 방법

### 1. Player 오브젝트 설정
1. Hierarchy에서 **Player** 선택
2. Inspector에서 **Add Component** 클릭
3. 다음 컴포넌트 추가:
   - `FlightSimulationController`
   - `MissileLauncher`

### 2. FlightSimulationController 설정
- **Planet Center**: 지구 오브젝트를 드래그 앤 드롭
- **Planet Radius**: 25 (자동 계산됨)
- **Move Speed**: 5
- **Dash Speed**: 15
- **Altitude**: 5 (초기 고도)

### 3. MissileLauncher 설정
- **Missile Prefab**: 비어있어도 됨 (자동 생성)
- **Fire Point**: 비어있어도 됨 (자동 생성)

### 4. Main Camera 설정
1. Hierarchy에서 **Main Camera** 선택
2. Inspector에서 **Add Component** 클릭
3. `CameraFollow` 컴포넌트 추가
4. **Target**: Player 오브젝트를 드래그 앤 드롭
5. **Offset**: (0, 3, -10)
6. **Prevent Flip**: 체크 (뒤집힘 방지)

## 조작 방법

- **W**: 앞으로 이동
- **S**: 뒤로 이동
- **A**: 왼쪽으로 이동
- **D**: 오른쪽으로 이동
- **Space**: 빠르게 앞으로 전진 (대쉬)
- **마우스 좌클릭**: 미사일 발사
- **마우스 우클릭 + 좌로 드래그**: 좌로 회전 (피하기)
- **마우스 우클릭 + 우로 드래그**: 우로 회전 (피하기)

## 특징

- 비행기는 지구를 기준으로 회전하며 이동합니다
- 카메라는 플레이어를 따라가며 뒤집힘 현상을 방지합니다
- 미사일은 비행기 앞쪽에서 발사됩니다
